What is Blender?

The Freedom to Create in 3D

Blender is a free and open-source 3D computer graphics software tool set used for creating animated films, visual effects, art, 3D-printed models, motion graphics, interactive 3D applications, virtual reality, and, formerly, video games

Molten Chrome Custom Typeface

This video teaches you how to create a unique chrome 3D font in Blender in the space of 20 minutes! This will teach you how to create 3D meshes from 2D designs as well as working with typefaces in the software.

Video Creator: Intranet Girl
Why I like this video?

This is a great introduction to using your own 2D designs or SVG files in Blender. You end up with a cool and trendy render at the end that's completely in-line with your own vision.

Create a City in Blender

This is a tutorial walks you through using the BLOSM Addon to create a 3D city in Blender based on real-life map data.

Video Creator: CG Geek
Why I like this video?

Creating 3D landscapes can be a chore, especially when creating large detailed planes - not only does this allow you to produce a cool environment based ona city you love but also gives you a basic understanding of how powerful Blender Add-Ons can be! It's very easy to follow and opens up a world (quite literally) of possibilities.
